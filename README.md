# ACM ICPC Website

Official Website for ACM ICPC

Built using React and Webpack. Uses [Chakra-UI](https://chakra-ui.com) for components.

## Setup

(**WIP:** Will be worked on later.)

### Install node modules

In root dirictory,
```
npm install
```

### Run dev server

In root directory,
```
npm run build-dev
```

will start the server on `localhost:3000`
